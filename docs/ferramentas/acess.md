# Acess Monitor Plus

Ferramenta oficial portuguesa para avaliação de acessibilidade, baseada nas WCAG 2.1 (ainda amplamente utilizada e atual). É reconhecida pela interface moderna e pela clareza dos relatórios visuais.

### Como usar

1. Acesse o site do Acess Monitor Plus.

2. Cole a URL da página.

3. Escolha o nível de conformidade desejado (A, AA ou AAA).

4. Analise o relatório gerado com marcações visuais e um relatório técnico detalhado.

### Pontos fortes

- Interface muito visual, fácil de entender até para iniciantes.

- Relatório técnico completo, com dicas e exemplos.

- Reconhecimento internacional, já que segue diretamente a WCAG.

## Pontos de atenção

- Mesmos limites de qualquer validador automático.

- Documentação e suporte pensados sobretudo para o contexto português.
