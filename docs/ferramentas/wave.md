# WAVE – Web Accessibility Evaluation  

Uma das ferramentas de acessibilidade mais conhecidas no mundo, desenvolvida pela WebAIM. É muito usada por designers, desenvolvedores e auditores.

## Como usar

1. Acesse o site e insira a URL, ou

2. Instale a extensão do Chrome/Firefox, que permite testar:

    - Páginas que precisam de login,

    - Ambientes de desenvolvimento (localhost),

    - Páginas dinâmicas.

## Pontos fortes

- Visualização direta na página, o que facilita entender o contexto do problema.

- Extensão poderosa para testar ambientes reais e dinâmicos.

- Muito educacional, ótimo para aprendizado.

- Credibilidade internacional.

## Pontos de atenção

- Pode "poluir" visualmente páginas muito complexas por exibir muitos ícones.

- Continua sendo automático — não detecta tudo.
