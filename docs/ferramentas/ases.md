# ASES – Avaliador de Acessibilidade de Sítios 

O ASES é o validador oficial de acessibilidade do governo brasileiro. Ele avalia páginas web com base no eMAG, que por sua vez se baseia nas WCAG, o padrão internacional mais usado.

## Como usar

1. Entre no site do ASES.

2. Insira a URL que deseja analisar.

3. Clique em "Avaliar".

4. O sistema gera um relatório com:

    - Erros (graves),

    - Avisos,

    - Itens que exigem verificação humana.

## Pontos fortes

- Ferramenta oficial do governo — garante conformidade com o eMAG.

- Relatórios didáticos, explicando o problema e como corrigir.

- Categorização clara (Erro/Aviso/Verificação Humana) facilita priorizar o que é mais urgente.

## Pontos de atenção

- Só faz testes automáticos — não consegue avaliar aspectos subjetivos, como a qualidade de uma descrição alternativa.

- Relatórios volumosos podem assustar quem está começando.
